# Titanic Survival Analysis

## 📌 Project Overview

This project explores the Titanic dataset to identify the factors that influenced passenger survival using Python and data analysis techniques.

The goal is to move through a complete data analysis workflow, from understanding the dataset to building predictive machine learning models.

---

## 🎯 Objectives

- Understand the dataset
- Perform Exploratory Data Analysis (EDA)
- Clean and prepare the data
- Visualize important trends
- Engineer useful features
- Build machine learning models
- Evaluate model performance
- Present meaningful business insights

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn *(to be added)*
- Scikit-learn *(to be added)*

---

## 📂 Dataset

The dataset used in this project is the Titanic dataset, which contains passenger information such as:

- Age
- Gender
- Passenger Class
- Fare
- Family Size
- Port of Embarkation
- Survival Status
